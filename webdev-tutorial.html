Web Development tutorial

By the end of this tutorial you will have a static website and unique domain name to host your personal website where you can show off all your all your traits!

We are going to host the website on Github. This amazing platform that allows version control of your code goes even further with their Github Pages support.

With a registered Github Account, follow this quick and simple guide from github to begin creating your site. Be sure to follow the steps for User or organization site. Return to the guide once you have completed those steps.

Great! Now you have a bare bones static site. Time to make it useful. Letâ€™s go back to the github pages site, but this time click on the tab for Project Site. Follow the instructions on how to generate a site in the repository you just created then come back to the guide.

Now you have a nice looking website with simple html and css code for you to customize! Notice the url of the site and how it has "github.io" at the end. As a student in college we can change that for free! Head over to the Namecheap for education site and claim a free domain to customize your site with. If you are not a college student, but are at a Major League Hacking sanctioned hackathon, you will be able to get a Domain.com domain name for free! Head on over to the MLH table or ask a representative to get a code for the free domain name.

Choose any domain name that represents the purpose of your website!

You are now able to use the domain to represent your website's address. Follow these instructions from Github to connect your github site with the domain name.

Congrats! You now have your own hosted static website. It is yours to hack with and develop some web dev skills. This is a great start to completing a web dev hack or if you just want to have a website to show off your projects (the webpage builder can be used for any repository on Github).
